# odin-recipes

## Description
This project is part of The Odin Project's Foundations course. It is a basic recipe website designed to practice foundational HTML skills before moving on to CSS and more complex web development concepts. The site consists of a main index page with links to several individual recipe pages.

## Skills Demonstrated
By completing this project, I have demonstrated the following skills:
*   **HTML Fundamentals:** Structuring web pages using boilerplate, headings, paragraphs, and lists.
*   **Navigation:** Implementing internal links to connect different HTML files within a project directory.
*   **Media Integration:** Adding images with descriptive alternative text.
*   **Version Control:** Using Git for tracking changes, committing work, and pushing code to a remote GitHub repository via SSH.
*   **Terminal Proficiency:** Navigating directories and managing files through the command line.
